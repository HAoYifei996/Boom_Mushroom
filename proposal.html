Proposal

Description

Due to our interests, we plan to create a simple FPS games based on Unity Engine and C#. It is the first time we create such project so it's a challenge for all of us. The development cycle is around 1 month and we present this project we called "Boom Mushroom".

Goals and Deliverables

In this project, we create a FPS game "Boom Mushroom"(BM). BM contains basic elements of a FPS game including online support and multiple access. BM supports up to 20 people in the same room to start a game. Additional materials such as first aid and room comunication are also provided(See Gallery for details).

What we plan to deliver:

  • A FPS game "Boom Mushroom"
  • Detailed development docs.

    • Schedule

    • Unity, Basic Environment Renderings

      1.0 week

      Download the basic Unity develop toolkit and environment setup. Expand on the previous knowledge of C#; Learn how to use C# in Unity and create an interactive application; Prepare basic environment settings for future development.

    • Game Scenes Setup

      1.0 week

      Deploy basic game scenes and character models in the Unity project. Create basic physical support(e.g walking, running) for game settings.

    • Game function development.

      1.5 week

      Add extra functions to the game to better reinforce gaming experience, detailed functions include HP bar, shooting gun fire, shooting touch etc.

    • Docs and web-pages

      0.5 week

      Detailed web pages provided, we decide to mix our docs into our presentation web pages. Web design and accomplishments presentation are done in this period.
